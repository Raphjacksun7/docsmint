## Project structure

Two directories matter. Everything else is generated.

```
your-project/
├── mkdocx.config.js
└── src/
    └── content/
        ├── docs/
        │   └── getting-started.md
        └── writing/
            └── first-post.md
```

`docs/` holds documentation pages. `writing/` holds dated posts. Both are optional — use whichever fits.

## Run locally

<Tabs labels={["pnpm", "npm", "pip"]}>
<Tab>
```bash

```
</Tab>
<Tab>
```bash

```
</Tab>
<Tab>
```bash

```
</Tab>
</Tabs>

Dev server starts at `http://localhost:4321`. Changes reload instantly.
